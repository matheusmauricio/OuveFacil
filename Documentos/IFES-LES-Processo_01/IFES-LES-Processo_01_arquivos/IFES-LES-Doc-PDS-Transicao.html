Documentação da Fase de Transição

 

 

 

 

 

Fase de transição do projeto. O sistema deve ser instalado e configurado no ambiente do cliente. A fase de Transição pode incluir a execução paralela de sistemas antigos e novos, migração de dados, treinamento de usuários e ajustes nos processos de negócio. Quando os objetivos da fase de Transição são alcançados, o projeto está pronto para ser encerrado.

 

 

 

 

 

12 Atividade: Implantar

Finalidade

  • Instalar o sistema no ambiente do cliente.
  • Realizar as configurações necessárias para utilização do cliente.
  • Preparar uma apresentação sobre como utilizar o sistema. Esta é, normalmente, uma apresentação realizada junto ao cliente, no sentido de facilitar a utilização do sistema.

Artefatos de Entrada:

  • Código Fonte – Alto Nível (CO3)

Artefatos de Saída:

 

·         SCV-MaquinaVirtual.vdi

·         SCV-Apresentacao_01.ppt

Papel: Programador

Ferramentas: Netbeans; VirtualBox; PowerPoint;

Como neste caso não existe o cliente real, vocês deverão realizar toda instalação do sistema em uma máquina virtual, utilizando a ferramenta VirtualBox.

  1. Instale a versão mais recente do VirtualBox.
  2. Crie uma máquina virtual, com o sistema operacional de sua preferência.
  3. Realiza as instalações e configurações necessárias para o funcionamento do sistema, por exemplo: instalação do Java, do banco de dados (se necessário), disponibilização do Jar (caso esteja utilizando Java) na área de trabalho, etc.
  4. Crie uma apresentação sobre como utilizar a ferramenta. Esta apresentação deve mostrar, pelo menos, a realização de uma funcionalidade de manutenção de cadastro, processo de negócio e listagem.
  5. Salve apresentação com o seguinte nome: <Sigla do Projeto>-Apresentacao_01.ppt

Observação: para sistemas Android, vocês deverão elaborar o arquivo de instalação do aplicativo.
