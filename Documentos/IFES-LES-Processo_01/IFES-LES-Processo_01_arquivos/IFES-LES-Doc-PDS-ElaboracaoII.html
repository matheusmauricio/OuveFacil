Finalidade

  • Gerar código em java automaticamente por meio da Ferramenta NetBeans. Com base no arquivo <Sigla do Projeto>-UML_03.asta

Artefatos de Entrada:

  • Diagramas de Classes de Projeto

Artefatos de Saída:

  • Código Fonte  Alto Nível (CO1)

Papel: Programador

Ferramentas: Astah; Netbeans

Passos:

  1. Crie um Projeto Aplicação Java

 

 

  1. Abrir o arquivo <Sigla do Projeto>-UML_03.asta com a ferramenta Astah Professional;
  2. Clique com em Tool >> Java >> Export Java...;

 

  1. Escolha a pasta src do Projeto criado anteriormente na Ferramenta NetBeans para geração do código;

  1. Clique em Select;
  2. Escolha o pacote model e, depois, selecione todas as classes destes pacotes;

  1. Verifique os códigos fontes.

a)      Todas as classes devem estar compilando (F9)

b)      Os imports necessários devem ser realizados (Ctrl + Shift + I)

c)      Implemente o método toString em todas as classes do pacote domain.

toString: este método retorna uma representação string do objeto. Em geral, o toString retorna uma string que "textualmente representa" esse objeto. O resultado deve ser uma representação concisa, mas informativa que é fácil para uma pessoa ler. Por exemplo:

Método toString da classe domain.Cliente

Artefatos de Saída:

Papel: Programador

Ferramentas: Word ou Writer; Netbeans

Passos:

  1. Ler Casos de Uso;
  2. Observar os Diagramas UML;
  3. Abrir o Projeto Java (Código Fonte  Alto Nível)
  4. Criar a interface gráfica para atender todos os passos identificados no Caso de Uso Expandido;
  5. Criar o pacote view;
  6. Criar JFrames (ou outro tipo de classe de interface gráfica, dependendo da tecnologia que esteja utilizando em seu projeto) no Pacote view;

a)      Crie um JFrame (ou outro tipo de classe) para Implementação da Tela Principal do Sistema

 

b)      Crie as demais telas a serem tratadas na atual iteração (conforme cronograma de projeto elaborado)
